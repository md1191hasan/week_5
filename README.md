# Week 5 Python Lab Tutorial

This folder contains the complete working files for Python Lab Tutorial 5.

## Files

- `main.py` - runs the program
- `utils.py` - contains the calculation and receipt functions
- `w5_documentation.md` - contains the documentation, process flow, constraints, decomposition, and pseudocode

## How to Run

Open the terminal inside this folder and run:

```bash
python main.py
```

If that does not work, run:

```bash
python3 main.py
```

## Sample Input

```text
Customer name: Bobby
Coffee quantity: 3
Tea quantity: 3
Sandwich quantity: 2
```

## Sample Output

```text
===== RECEIPT =====
Customer : Bobby
Coffee   : 3
Tea      : 3
Sandwich : 2
------------------
Total = RM 67.50
```
